wmaps.org

wmap.org is a secure, permission-based web application that displays a dynamic world map where authenticated users can add pins, flags, and visual markers. The project focuses on creating a system similar to Liveuamap, but with controlled access, user authentication, and backend security.

The backend is built with Node.js, Express, MongoDB, and JWT authentication, and supports role-based permissions for map editing. The app is designed for fast loading, safe data handling, and a clean UI.

This project reflects my growing skills in backend development, secure authentication, and modern web technologies.

  • Full-Stack Web Development
  • Node.js
  • MongoDB
Project Link
Lumina — Advanced Vocabulary Builder

Lumina is an Android vocabulary-building app designed to help users expand their English vocabulary through quizzes, custom word lists, and practice tracking. Built in Android Studio (Java) with an SQL database, the app features a clean UI, a practice timer, user login system, and a Home screen that displays the last 5 practice sessions in a single CardView.

Lumina focuses on simplicity, daily progress, and helping users learn words efficiently. It is still in development and will be released publicly in the future.
